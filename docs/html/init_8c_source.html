48 /*
49  * Runs user initialization code. This function will be started in its own task with the default
50  * priority and stack size once when the robot is starting up. It is possible that the VEXnet
51  * communication link may not be fully established at this time, so reading from the VEX
52  * Joystick may fail.
53  *
54  * This function should initialize most sensors (gyro, encoders, ultrasonics), LCDs, global
55  * variables, and IMEs.
56  *
57  * This function must exit relatively promptly, or the operatorControl() and autonomous() tasks
58  * will not start. An autonomous mode selection menu like the pre_auton() in other environments
59  * can be implemented in this task if desired.
60  */
