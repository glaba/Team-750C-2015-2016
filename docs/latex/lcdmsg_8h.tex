\subsubsection{Detailed Description}
Header file for L\+CD message code. 

This file contains definitions for the L\+CD screensaver messages. This file also defines the L\+CD port. These messages display randomly while the L\+CD diagnostic menu is set to the screensaver mode. These messages are mainly inside jokes among the team.
