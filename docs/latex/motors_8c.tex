\subsubsection{Detailed Description}
File for important motor functions. 

This file contains the code for functions regarding motor status. These functions are too complex to be defined as inline functions in the \hyperlink{motors_8h}{motors.\+h} file.

See the \hyperlink{motors_8h}{motors.\+h} file for the basic movement functions.
