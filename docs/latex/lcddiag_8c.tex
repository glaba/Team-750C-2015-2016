\subsubsection{Detailed Description}
File for L\+CD diagnostic menu code. 

This file contains the code for the L\+CD diagnostic menu. The menu provides live debugging and testing functionality. It provides the following functions\+:
\begin{DoxyItemize}
\item Motor testing functionality (individual and group)
\item Motor group management
\item Battery voltage information
\item Joystick connection status
\item Robot sensory data
\item Autonomous recorder status
\item L\+CD backlight toggle
\item Screensaver that displays during operator control
\item Credits menu
\end{DoxyItemize}

The idea behind this was inspired by Team 750W and Akram Sandhu. Without them, this project would not be possible.

Note\+: the implementation of this feature is completely different between the two teams. No code was reused from their implementation of the L\+CD diagnostic menu.
