\subsubsection{Detailed Description}
Header file for human-\/readable definitions. 

This file provides constant definitions for various sensory states that appear frequently. Since the button pressed state is represented by an unintuitive L\+OW value, these constants create definitions for the pressed and unpressed states of a button.
